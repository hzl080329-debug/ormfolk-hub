# ORMFOLK Hub — 需求规格说明

## 项目信息
- **项目名称**：ORMFOLK Hub
- **定位**：全球 ORMFOLK 粉丝免费社区网站
- **主角**：Supitcha Limsommut（Ormsin / ออมสิน）& Folk Sutima Kokiatwanit（Folk / โฟล์ค）
- **CP 名称**：OrmFolk（中文昵称：奥福）
- **目标用户**：全球 ORMFOLK 粉丝（新粉 + 老粉）

## 用户需求
1. 支持电脑端和手机端访问（响应式设计）
2. 支持英文、中文、泰文三种语言
3. 用户注册登录（邮箱 + Google/X 社交登录）
4. 普通用户 + 管理员双角色

## 功能模块（8 个）

### 1. 首页动态
- 最新帖子、热门内容、官方公告聚合展示
- 瀑布流/卡片式布局，支持筛选（全部/公告/热门）

### 2. ORMFOLK 时间线与作品资料库
- 纵向时间轴展示 Ormsin & Folk 成长历程
- 作品资料库（影视、音乐、活动等）
- 粉丝投稿 → 管理员审核 → 发布
- 按年份、类型、人物筛选

### 3. 社区论坛
- 分区讨论（新粉入门、剧集讨论、CP 时刻、闲聊等）
- 发帖、评论、点赞

### 4. 粉丝创作区
- 上传作品（同人文、绘画、视频、手工）
- 瀑布流展示，按类型筛选

### 5. 活动日历
- 月历视图 + 列表视图
- 管理员添加活动、自动抓取社交媒体
- 粉丝标记「感兴趣」/「参加」

### 6. 全球粉丝地图
- Leaflet 世界地图
- 粉丝手动选择城市，城市级别显示
- 隐私：可设置隐藏

### 7. 留言墙
- 祝福留言、入坑故事
- 便签墙风格，支持 emoji、点赞、回复

### 8. 用户个人主页
- 个人资料、发帖记录、创作作品集、收藏夹
- 编辑资料、语言偏好、隐私设置

## 设计风格
- **主色调**：薰衣草紫 `#C8A2D6` / `#C9B1FF`、柔和粉色 `#F4C2C2` / `#FFD1E0`、白色 `#FFFFFF`
- **辅助色**：深紫 `#7B5EA7`、浅灰 `#FAFAFA`
- **字体**：Nunito（英文）、Noto Sans SC（中文）、Noto Sans Thai（泰文）
- **风格**：温暖、梦幻、现代、简洁

## 非功能需求
- 免费使用
- 长期保存 ORMFOLK 成长历程和粉丝回忆
- 新粉快速了解、老粉有归属感
