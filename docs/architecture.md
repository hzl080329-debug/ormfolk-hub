# ORMFOLK Hub — 系统架构说明

## 整体架构

```
┌─────────────────────────────────────────────────┐
│                   用户浏览器                       │
│          (Desktop / Mobile / Tablet)             │
└─────────────────────┬───────────────────────────┘
                      │
┌─────────────────────▼───────────────────────────┐
│               Next.js 16 (App Router)             │
│                                                   │
│  ┌──────────┐  ┌──────────┐  ┌───────────────┐  │
│  │ 页面组件  │  │ API 路由  │  │ Server Actions │  │
│  │ (RSC)    │  │ (REST)    │  │ (表单处理)     │  │
│  └──────────┘  └──────────┘  └───────────────┘  │
│                      │                            │
│  ┌───────────────────▼──────────────────────────┐│
│  │              Prisma ORM                      ││
│  └───────────────────┬──────────────────────────┘│
└──────────────────────┼───────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────┐
│                  数据库                           │
│        SQLite (dev) → PostgreSQL (prod)          │
└─────────────────────────────────────────────────┘
```

## 路由架构

```
/                           → 根页面 → 重定向到 /en
/[locale]/                  → 首页（en/zh/th）
/[locale]/timeline          → 时间线 + 资料库
/[locale]/forum             → 论坛首页（分区列表）
/[locale]/forum/[slug]      → 分区帖子列表
/[locale]/forum/[slug]/[id] → 帖子详情
/[locale]/creations         → 粉丝创作区
/[locale]/creations/[id]    → 创作详情
/[locale]/calendar          → 活动日历
/[locale]/map               → 全球粉丝地图
/[locale]/wall              → 留言墙
/[locale]/user/[id]         → 用户个人主页
/[locale]/admin             → 管理后台
/[locale]/login             → 登录/注册
```

## 数据流

1. **页面加载**：Server Component → Prisma 查询 → 渲染 HTML → 发送客户端
2. **用户交互**（发帖等）：Client Component → Server Action / API Route → Prisma → 数据库 → revalidate
3. **认证**：NextAuth.js 中间件 → JWT Session → 页面/API 鉴权

## 国际化架构

- `next-intl` 路由级国际化
- 翻译文件：`messages/{locale}.json`
- 中间件自动检测/切换语言
- 语言优先级：URL 路径 > Cookie > 浏览器设置 > 默认英文

## 认证流程

```
用户注册 → 邮箱 + 密码 → 邮箱验证 → 登录
用户注册 → Google → OAuth 回调 → 自动创建账号 → 登录
管理员 → 由其他管理员在后台提升权限
```
