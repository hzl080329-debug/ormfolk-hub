## 🚀 待你下一步操作

### 1. 替换真实照片
在 `public/images/` 目录下放入 Ormsin 和 Folk 的照片：
- `ormsin.jpg` — Ormsin 的照片
- `folk.jpg` — Folk 的照片

### 2. 配置 Google OAuth（可选）
1. 去 https://console.cloud.google.com 创建 OAuth 凭证
2. 将 Client ID 和 Secret 填入 `.env` 的 AUTH_GOOGLE_ID 和 AUTH_GOOGLE_SECRET

### 3. 部署上线（可选）
```bash
npm run build   # 确认构建成功
# 然后部署到 Vercel（免费）或购买服务器
```
- 域名建议：ormfolk-hub.com ($10-15/年)
- 托管：Vercel（免费层足够起步）
